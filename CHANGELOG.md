# Changelog

All notable changes to Versionx are documented here. Format loosely follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/); version numbers follow [SemVer](https://semver.org/) (strict from 1.0; loose during 0.x).

The changelog itself is eventually generated by Versionx (dogfood). Until then, it's hand-maintained.

## [Unreleased]

### Added
- Initial repository scaffold: Cargo workspace with 27 member crates, CI pipeline, release tooling (cargo-dist), license headers, CODEOWNERS, contributor docs, security policy.
- Full vision specification in `docs/spec/` (12 documents covering architecture, config, adapters, runtimes, release orchestration, multi-repo, policy, CI, APIs, MVP, and the per-version 0.1 → 1.4 roadmap).

### Notes
- Binary name is **`versionx`** (not `vx`). `vx` is taken on crates.io by an active tool in the same space; `versionx` is free across all registries.
- License: Apache-2.0.
- MSRV: rustc 1.95 (latest stable at scaffold time).
- Project-wide zero-telemetry stance: no network call ever fires without explicit user action.
