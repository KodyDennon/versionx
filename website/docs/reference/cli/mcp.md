# `versionx mcp`

MCP server for AI-agent integration

## Subcommands

### `serve`

Serve the MCP protocol over stdio (the default for agent integrations)

#### Flags

| Flag | Value | Default | Required | Global | Description |
|---|---|---|---|---|---|
| `--http` | `<PORT>` |  |  |  | Listen on a loopback HTTP port instead of stdio. The server binds 127.0.0.1:`PORT` and uses rmcp's streamable-http transport with default DNS-rebind protection (loopback only) |
