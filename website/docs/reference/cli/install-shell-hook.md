# `versionx install-shell-hook`

Auto-install the activation hook into the user's shell rc file (~/.zshrc, ~/.bashrc, etc.). Idempotent: re-running detects the existing marker + skips

## Flags

| Flag | Value | Default | Required | Global | Description |
|---|---|---|---|---|---|
| `--shell` | `<SHELL>` |  |  |  | Force a specific shell. Defaults to `$SHELL`. Possible values: `bash`, `zsh`, `fish`, `pwsh`. |
