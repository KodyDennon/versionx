# `versionx activate`

Emit a shell integration hook. Put `eval "$(versionx activate bash)"` in your shell rc file to start the daemon on login and prepend the shims dir to `PATH`

## Flags

| Flag | Value | Default | Required | Global | Description |
|---|---|---|---|---|---|
| `shell` | `<SHELL>` |  | yes |  | Which shell's hook to emit. Possible values: `bash`, `zsh`, `fish`, `pwsh`. |
